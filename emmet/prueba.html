Ejemplos Elementos

Anidar Elementos

Elementos Hermanos

section+header+nav
<section></section> <header></header> <nav></nav>

Repetir Elementos (*)

ul>li*3
<ul> <li></li> <li></li> <li></li> </ul>

Atributo Clase

div.contenedor

.contenedor2

div.clase1.clase2.clasen

nav#menu
<nav id="menu"></nav>
